# Employee Management System (Java OOP)

A desktop application designed to manage corporate hierarchies and employee data using Object-Oriented Programming principles.

## 🛠️ Key Technical Focus
* **OOP Fundamentals:** Implemented Inheritance, Encapsulation, and Polymorphism to manage different roles.
* **Data Structures:** Used Java Collections (ArrayList/HashMap) to manage and search through records efficiently.
* **UI/UX:** Built a functional interface for data entry and visualization.

## 🧠 Why this project?
I built this to master the Internal Logic of an application before moving into Database persistence. It helped me understand how to structure complex data within a Java environment.
